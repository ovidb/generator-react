A [yeoman](yeoman.io/) generator for react

Usage:

```
npm i -g yo @ovidb/generator-react
yo @ovidb/react 

? Location : ./components
? Component category : general
? Your component name : component

    create components/general/component/component.jsx
    create components/general/component/component.story.js
    create components/general/component/component.test.js
    create components/general/component/component.module.scss                      
```
